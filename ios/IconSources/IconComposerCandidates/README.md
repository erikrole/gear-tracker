# Icon Composer Source

`vintage-helmet/BlockW.icon` is the manually finalized and only approved Wisconsin Creative app icon source.

- Default appearance: white Block W on Wisconsin red.
- Dark appearance: Wisconsin-red Block W on black.
- Validate Default, Dark, and Mono at both Home Screen and 1024-point sizes.
- Never bake an app-icon corner radius into the SVG.

The wired copy lives at `ios/Wisconsin/AppIcons/AppIcon.icon`. After any manual Icon Composer edit, copy the complete `.icon` package again; never merge only `icon.json` or only its `Assets` directory.
