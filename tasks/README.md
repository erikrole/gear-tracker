# Tasks Folder

Use this folder for active planning and review notes.

- Keep current execution ledgers at the root, such as `todo.md`, `lessons.md`, and active `*-plan.md` files.
- Move completed plan files to `tasks/archive/` or a dated subfolder under it.
- Keep screenshot and browser-smoke proof artifacts in `tasks/archive/proofs/`, not in the root task folder.
- Do not delete completed plans; archive them so later cleanup can reconcile why a slice moved.
